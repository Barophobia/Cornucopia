Networking
What is networking?

Networking is the communication on a network between devices.


What is DHCP?

DHCP (Dynamic Host Configuration Protocol) is used to dynamically assign devices on the network IP addresses automatically, an easy way to understand this is if you imagine a router in a household 99% of homes most likely are just using the default configuration of the router and when you join the network you are automatically given an IP address, the router is giving you that IP address on a lease meaning that if the lease ends and has not been renewed automatically by your device then the IP address will be put back in the DHCP scope and handed to another device.


What is a static IP?

A static IP is an IP Address that does not change after being set for example on a /24 network you could set the IP address of a device to 192.168.0.2 where the router would be 192.168.0.1 and the device would hold that IP until you change it, if you are using static IP's on a network then you will need to take the static IP's out of the DHCP scope.


Setting a static IP address in CMD or PS:

PS: 1) Get-NetAdapter This will give you the ifindex number of the interfaces in your machine
2) New-NetIPAddress -InterfaceIndex *ifindex number* -IPAddress *required static IP Address -PrefixLength *SubnetMask ex. 24* -DefaultGateway *DefaultGateway IP*
This will set the static IP in powershell
CMD: 1) Netsh interface ipv4 set address name="*Name of interface*" static *IP address* *subnet* *default gateway* This will set the static IP address using CMD.
