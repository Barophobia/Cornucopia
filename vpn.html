Virtual Private Network

What is a VPN? (Basic Explanation)

A VPN (Virtual Private Network) is when you connect to a remote site/server and route traffic through the remote connection. This could be used in a corporate setting where you can get a connection to the company network to access the intrantet or it could be used in a personal setting where you want to change your IP addresses 'location'.

How does the VPN connect to the remote site?

When connecting to a VPN it uses a tunnel, the data you send is then encrypted and securely encapsulated and sent using TCP/IP (explained here).

Split Tunneling

Split tunneling allows you to route some of your packets through the VPN tunnel as well as using your usual ISP for normal internet activities. This could be used to save bandwidth on the network you are tunneling into to make sure that other people using the VPN have the bandwidth needed for their tasks as well.

Checking Cisco active VPN users

To check active VPN users on a cisco router through the CLI you can use:

show crypto session detail

This will give you the info about the clients and their session.
