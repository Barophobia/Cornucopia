Windows ADK
What is ADK?

Windows Assessment and Deployment Kit (Windows ADK), formerly Windows Automated Installation Kit (Windows AIK or WAIK), is a collection of tools and technologies produced by Microsoft designed to help deploy Microsoft Windows operating system images to target computers or to a virtual hard disk image in VHD format.

What tools are in the ADK?

The ADK is currently made up of 9 tools which break down into many more, they are as follows:

Application compatibility tools: This toolset consists of Compatibility Administrator and Standard User Analyzer.
DISM (Deployment Image Service and Management Tool): This is a tool introduced in Windows 7 and Windows Server 2008 R2 that can perform servicing tasks on a Windows, be the operating system running the system or an offline image within a folder or WIM file. Its features include mounting and unmounting images, querying installed device drivers in an offline image, and adding a device driver to an offline image.
Imaging and configuration designer: This allows you to create, edit and manage the image that you want to deploy as well as the software and settings inside the image (like WDS/MDT).
WSIM (Windows System Image Manager): This allows you to use a GUI to create the unattend.xml file to automate deployments.
Windows assesment toolkit: This allows you to assess computers to see if you are going to have issues with deployment.
Windows performance toolkit: This tool allows you to analyse performance on a device and manage aswell.
USMT (User state migration tool): This tool allows you to gather data and settings from a user and migrate them to a new deployment.
VAMT (Volume Activation Management Tool): This tool allows you to automate the activation of microsoft products for clients on the network.
UEV (User Experience Virtualization): This monitors the changes users make to application settings and allows them to follow the user on multiple devices.
App-V (Application Virtualization): This runs applications at a remote location and allow the user to use it locally.
